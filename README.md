# Boilerplate Typescript 

This Boilerplate contains a basic template for the frontend and the backend built in Typescript 

## Table of Contents

#### Front-End  ( /client ) 
> Build with npx create-react-app my-app --template typescript

Extra :
- Redux
- Material UI
     
#### Back-End   ( /api )  

Build with : 

* Express
* Prisma (ORM)
